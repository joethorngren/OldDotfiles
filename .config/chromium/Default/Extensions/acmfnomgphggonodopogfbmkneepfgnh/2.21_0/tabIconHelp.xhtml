Bookmark Favicon Changer - Tab Icon Rule Help

  • Tab icon rules are used to change icons for the whole domain or sub domain.
  • URL that can use must begin with http, https or file.
    • Wildcard * in the rule is accepted as a usual meaning except one thing.
    • If * is not at the last position, it will represent any character except "/" at any length.
    • But if * is at the last position, the meaning is the same as usual (any character at any length).
    • Example...
      • http*://www.example.com/*
      • This will represent the whole domain of www.example.com including https and http.
      • http//*.google.com/*
      • This will match http://mail.google.com but not match http://google.com and not match http://www.yahoo.com/search?mail.google.com
  • You can change tab icon to your custom icon by click the icon at the left most of the row.
  • Any size and type of image files can be use as custom tab icon. "Bookmark Favicon Changer" will resize and change it to 16x16 pixel PNG image file and then save. After changing icon, there is no need for original file to exist.
  • Tab icon rule only applied when there is no match in custom favicon database for bookmark.
  • Ordering of tab icon rules is also important. The first rule that match will be use as tab icon. You can drag-drop the rule to change the rule ordering.
  • Despite you can use tab icon rule to change bookmark favicon, it is not recommended. The bookmark favicon protection mechanism is weaker.
