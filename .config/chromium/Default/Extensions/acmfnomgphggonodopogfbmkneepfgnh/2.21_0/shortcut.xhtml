For some bookmarks that are difficult to change favicon, you can use the "shortcut technique". "Bookmark Favicon Changer" will create a new bookmark. This bookmark behaves like a shortcut. If you click it, it will immediately redirect to your target URL. But the advantage is... it has your own custom favicon. Your target URL can be http, https, ftp, file, etc...(Just try it). This shortcut is very special, it does not need "Bookmark Favicon Changer" to be active and it does not create any file on your hard drive. It will work forever just like a normal bookmark.

Step 1: Input full target URL (e.g. http://www.google.com).
Step 2: Input bookmark name (optional)
Step 3: Select custom favicon
Step 4: Test redirect to target URL
Step 5: Choose where to add shortcut
 First bookmark in "Bookmarks Bar"     First bookmark in "Other Bookmarks" folder
 Last bookmark in "Bookmarks Bar"     Last bookmark in "Other Bookmarks" folder
Step 6: Add shortcut
