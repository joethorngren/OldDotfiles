Bookmark Favicon Changer Help

  • Right click at bookmark to manipulate favicon.
  • "Change favicon" - changing favicon of selected bookmark. You can change favicon only one by one and URL must begin with http, https or file. Any size and type of image files can be use as custom favicons. "Bookmark Favicon Changer" will resize and change it to 16x16 pixel PNG image file and then save in database. After changing favicon, there is no need for original file to exist.
  • "Pin favicon" - use current favicon as a custom favicon. This will prevent website to change this favicon.
  • "Reset favicon" - revert favicon back to original one or try to load from the internet at the most likely address.
  • "Remove favicon" - remove favicon and use default one.
  • "Export favicon" - save favicon in your hard disk.
  • You can export any favicon. But you can only "change favicon", "reset favicon" and "remove favicon" from URL that begin with http, https or file.
  • A small window will appear during change, reset or remove favicon. This is normal and needed.
  • If you have difficulty to change or protect your custom favicon, please try use "Shortcut Technique". It is very special and more powerful.
  • You need "Bookmark Favicon Changer" to be active to prevent favicon turn back. You can't just change favicon and uninstall extension. It will not work, favicon will turn back when you visit that page or you clear the history or favicon reach expire date. But the good news is, "Shortcut technique" has no these limitations.
  • There are more things you can do by click "More" or "Advance Settings."
