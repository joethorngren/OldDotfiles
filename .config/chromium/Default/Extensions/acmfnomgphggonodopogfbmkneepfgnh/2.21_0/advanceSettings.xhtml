Page Action is an icon that show on URL bar (see picture on the right). You can choose when to show "Page Action".

  This icon will show on page that use custom favicon or tab icon.

  This icon will show on page that don't use custom favicon or tab icon but it can be changed to use it.

Default is "show on page that can use custom favicon".
You can also change the on click event by choosing to "open popup" (default) or directly "open the options page".
